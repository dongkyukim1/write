# 스크립트 사용 가이드

대본 작성을 도와주는 유틸리티 스크립트들입니다.

## generate_script.py

매불쇼 대본의 각 섹션을 AI로 생성하는 대화형 도우미입니다.

### 사용 방법

```bash
# 프로젝트 루트에서 실행
python scripts/generate_script.py
```

### 기능

1. **오프닝 생성**: 이번 주 이슈를 바탕으로 오프닝 멘트 생성
2. **뉴스 요약**: 핵심 사실을 입력하면 뉴스 요약 섹션 생성
3. **토크 섹션**: 주제를 입력하면 본격 토크 섹션 생성
4. **하이라이트**: 핵심 포인트를 입력하면 하이라이트 구간 생성

### 예시

```bash
$ python scripts/generate_script.py

==================================================
매불쇼 대본 생성 도우미
==================================================

✅ OpenAI 사용

대본 정보를 입력해주세요:
--------------------------------------------------
에피소드 제목: 이번 주 국회 이슈
방송일 (YYYY-MM-DD): 2024-12-23
메인 토픽: 국회 관련 주요 뉴스

오프닝 생성 중...
✅ 오프닝 생성 완료
```

### 주의사항

- `.env` 파일에 API 키가 설정되어 있어야 합니다
- OpenAI 또는 Claude API 키 중 하나는 필수입니다
- 생성된 내용은 참고용이며, 작가가 수정/보완해야 합니다
