## 📁 폴더 구조

```
examples/
├── talk_show/          # 방송 대본 예제
```

## 🎯 사용 방법

1. 각 폴더의 예제 파일을 참고하세요
2. 자신의 프로젝트에 맞게 수정하여 사용하세요
3. 새로운 예제를 추가하여 공유해주세요

## 📝 예제 추가하기

새로운 예제를 추가할 때는:
1. 적절한 폴더에 파일 생성
2. 파일명은 `example_[설명].md` 형식
3. 사용된 프롬프트/템플릿 명시
4. 변수 값과 결과 포함
