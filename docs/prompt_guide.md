# 프롬프트 작성 가이드

효과적인 AI 프롬프트를 작성하는 방법을 안내합니다.

## 기본 원칙

### 1. 명확성 (Clarity)
- 모호한 표현보다는 구체적인 지시 사용
- 예: "좋은 소설" ❌ → "SF 장르의 3000자 단편 소설" ✅

### 2. 구조화 (Structure)
- 단계별로 나누어 작성
- 체크리스트나 번호 목록 활용

### 3. 예시 제공 (Examples)
- 원하는 결과의 예시 포함
- 스타일이나 형식의 참고 자료 제공

### 4. 제약 조건 명시 (Constraints)
- 길이, 형식, 금지 사항 등을 명확히

## 프롬프트 템플릿 구조

```markdown
# [제목]

## 역할 정의
당신은 [역할]입니다.

## 작업 설명
[구체적인 작업 내용]

## 입력 정보
- 변수1: {{variable1}}
- 변수2: {{variable2}}

## 요구사항
1. [요구사항 1]
2. [요구사항 2]

## 출력 형식
[원하는 출력 형식]

## 예시
[예시 내용]
```

## 변수 사용법

프롬프트에서 변수는 `{{변수명}}` 형식으로 사용합니다.

```markdown
{{genre}} 장르의 {{length}} 소설을 작성해주세요.
```

사용 시:
```python
prompt = load_prompt_template(
    "story/novel_generator.md",
    genre="SF",
    length="단편"
)
```

## 좋은 프롬프트 vs 나쁜 프롬프트

### 나쁜 예시 ❌
```
소설 써줘
```

### 좋은 예시 ✅
```
당신은 전문 소설가입니다. 
SF 장르의 단편 소설을 작성해주세요.

요구사항:
- 길이: 3000자 내외
- 주제: 시간여행
- 스타일: 모던하고 빠른 템포
- 첫 문장은 독자의 관심을 즉시 끌어야 함

출력 형식: 마크다운
```

## 팁

1. **역할 부여**: AI에게 명확한 역할을 부여하세요
2. **단계별 지시**: 복잡한 작업은 단계로 나누세요
3. **피드백 루프**: 결과를 보고 프롬프트를 개선하세요
4. **템플릿 재사용**: 효과적인 프롬프트는 템플릿으로 저장하세요
