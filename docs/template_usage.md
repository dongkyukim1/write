# 템플릿 사용법

템플릿을 활용하여 효율적으로 작품을 작성하는 방법을 안내합니다.

## 템플릿이란?

템플릿은 반복적으로 사용할 수 있는 구조화된 문서 형식입니다.
- 체크리스트 포함
- 구조화된 섹션
- 가이드라인 제공

## 템플릿 사용 절차

### 1. 템플릿 선택
작성하려는 작품 유형에 맞는 템플릿 선택:
- 방송 대본: `templates/talk_show/`
- 소설: `templates/novel/`
- 시나리오: `templates/script/`

### 2. 템플릿 복사
원본 템플릿을 복사하여 새 파일 생성:
```bash
cp templates/talk_show/maebul_show_template.md my_episode.md
```

### 3. 각 섹션 채우기
템플릿의 각 섹션을 순서대로 채워나갑니다:
- 기본 정보 입력
- 각 섹션 내용 작성
- 체크리스트 확인

### 4. 검토 및 수정
- 러닝타임 체크
- 티키타카 자연스러운지 확인
- 누락된 부분 확인

## 방송 대본 템플릿 사용 예시

### 매불쇼 템플릿 사용법

1. **에피소드 정보 입력**
   ```
   에피소드 제목: 이번 주 국회 이슈
   방송일: 2024-12-23
   주제: 국회 관련 뉴스
   ```

2. **오프닝 작성**
   - 최욱 멘트 작성
   - 정영진 반응 작성
   - 자연스러운 전환 고려

3. **뉴스 요약 섹션**
   - 핵심 사실 3-4개 정리
   - 정영진 반응 포인트 기획

4. **본격 토크**
   - 섹션별로 주제 나누기
   - 티키타카 포인트 미리 기획
   - 웃음 포인트 배치

5. **체크리스트 확인**
   - 모든 항목 체크
   - 누락된 부분 보완

## 템플릿 커스터마이징

자신만의 템플릿을 만들려면:

1. 기존 템플릿을 참고하여 새 템플릿 생성
2. 자신의 작업 방식에 맞게 수정
3. `templates/` 폴더에 저장

## 팁

- **템플릿은 가이드라인**: 고정된 구조보다는 참고용으로 활용
- **유연하게 사용**: 필요에 따라 섹션 추가/삭제
- **반복 작업 자동화**: 자주 사용하는 부분은 스크립트로 자동화
