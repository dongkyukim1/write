## 📋 빠른 체크리스트

시작하기 전에 확인하세요:

- [ ] Python이 설치되어 있나요?
  - 확인: `python --version`
  - 없으면: [초보자 가이드](GUIDE_FOR_BEGINNERS.md) 참고

- [ ] 필요한 패키지가 설치되어 있나요?
  - 확인: `pip list`
  - 없으면: `pip install -r requirements.txt`

- [ ] API 키가 준비되어 있나요?
  - OpenAI: https://platform.openai.com
  - Claude: https://console.anthropic.com
  - 설정: `.env` 파일에 저장

- [ ] 테스트가 성공했나요?
  - 실행: `python scripts/test_api.py`
  - 성공 메시지가 나오면 준비 완료!
